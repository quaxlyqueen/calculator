# A calculator
A calculator written in Python for the GUI and using eva for basic computation.

# Installation:
`git clone https://github.com/quaxlyqueen/calculator`<br>
`cd calculator`<br>
`sudo ./install`<br>

# Uninstall:
`sudo ./uninstall`<br>
